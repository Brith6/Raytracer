Detailed Description

Classe principale du moteur de ray tracing.

Cette classe gère le lancement du programme, la lecture des arguments en ligne de commande, l'affichage de l'aide, le parsing de la scène, et le rendu final de l'image.

int core::RayTracer::run ( ) const

Lance l'exécution principale du programme.

Exécute le programme principal de ray tracing.

Traite les arguments, charge et parse le fichier scène, initialise la scène, effectue le rendu et affiche ou sauvegarde l'image.

Returns
int Code de retour (0 en succès, autre en erreur).

Traite les arguments en ligne de commande, ouvre et parse le fichier scène, initialise la scène et l'image, puis lance le rendu. Peut afficher soit une fenêtre SFML si l'option -g est passée, soit sauvegarder une image PPM.

Exceptions
raytracer::ArgumentErrorsi aucun fichier scène n'est fourni.
raytracer::ParseErrorsi le fichier scène ne peut pas être ouvert ou parsé.
Returns
int Code de retour du programme (0 en succès, 84 en erreur).
