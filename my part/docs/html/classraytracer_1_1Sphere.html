Detailed Description

Représente une sphère dans la scène 3D pour le moteur de raytracing.

Cette classe hérite de IPrimitive et implémente les méthodes pour gérer l'intersection avec un rayon ainsi que la normale à un point donné.

Calcule l'intersection entre un rayon et la sphère.

Parameters
rayLe rayon lancé dans la scène.
Returns
La distance entre l'origine du rayon et le point d'intersection, ou -1.0 s'il n'y a pas d'intersection.

Utilise l'équation quadratique pour déterminer si le rayon touche la sphère. Si le discriminant est négatif, il n'y a pas d'intersection.

Parameters
rayLe rayon lancé dans la scène.
