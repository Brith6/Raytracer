Detailed Description

Gère la création d'objets (primitives et lumières) dans le moteur de raytracing.

Cette classe utilise le pattern de conception Factory pour instancier dynamiquement des objets héritant des interfaces IPrimitive et ILight, en fonction de leur type.

raytracer::Factory::Factory ( )

Constructeur par défaut.

Constructeur de la classe Factory.

Initialise les types de primitives et de lumières en les enregistrant.

Ce constructeur enregistre tous les types de primitives (objets 3D) et de lumières supportés par le moteur. Il utilise des templates pour associer chaque type à son constructeur.
