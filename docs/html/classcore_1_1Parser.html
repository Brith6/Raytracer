Detailed Description

Classe responsable du parsing des fichiers de configuration pour la scène 3D.

La classe Parser analyse les fichiers de configuration (libconfig) pour extraire les paramètres de la caméra, les primitives (sphères, plans, cylindres, cônes), ainsi que les lumières directionnelles et ponctuelles. Elle stocke ensuite ces objets dans des collections internes accessibles via des getters.
